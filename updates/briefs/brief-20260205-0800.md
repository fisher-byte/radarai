## 🔍 核心洞察  
**Gemini** 月活用户突破 **7.5 亿**，API 每分钟处理 **100 亿 Token**；**OpenAI Codex** 正式集成 GitHub Agent HQ 并达成 50 万次下载；行业范式加速转向 **Agentic Engineering** 与**商业实用性优先**的评估新标准。

## 🚀 重点动态  
- **ChatGPT 全面支持 MCP Apps 标准**：实现跨平台模型上下文协议（MCP）互操作性，推动 AI 应用生态标准化。  
- **Google Gemini 月活达 7.5 亿，Token 吞吐量破 100 亿/分钟**：Jeff Dean 确认该数据为当前全球最大规模 AI 服务实时负载指标。  
- **OpenAI Codex 正式上线 GitHub Agent HQ**：面向 Copilot Pro+ 与 Enterprise 用户开放，深度嵌入开发工作流自动化。  
- **Artificial Analysis 智能指数 v4.0 发布**：基准测试重心转向商业实用性与真实场景推理能力，GPT-5.2 首登榜首。  
- **Perplexity 开源 DRACO 基准测试与数据集**：聚焦法律、医疗等高价值垂直领域，提供可复现的 SOTA 评估框架。  
- **LangSmith 推出全新并排实验对比视图**：支持多模型、多提示、多链路横向性能追踪，显著提升 LLM 应用迭代效率。  
- **Milvus 集成 Anthropic Skills**：用户可通过自然语言指令一键生成 RAG 向量数据库流水线，降低检索增强门槛。  
- **LlamaParse 推出「Agentic Plus」模式**：基于 VLM 实现图表/流程图→Mermaid 的高精度自动解析，强化文档智能体基建能力。
