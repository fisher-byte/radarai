## 🔍 核心洞察  
**Kimi K2.5** 成为全球基础设施级AI部署新标杆——Cloudflare 在核心生产业务中采用该模型，实现 **77% 成本降幅** 同时支撑 AI Agent 与自动化代码审查 [19]；与此同时，**IBM Granite 4.0 3B Vision** 以模块化 **DeepStack 架构** 和专有 **ChartNet 数据集** 突破企业文档理解瓶颈，凸显多模态轻量化落地加速趋势 [0]。

## 🚀 重点动态  
- **IBM 发布 Granite 4.0 3B Vision 多模态模型** [0]：面向表格提取与图表理解优化的紧凑型视觉语言模型，采用 DeepStack 架构  
- **Cloudflare 全面接入 Kimi K2.5** [19]：用于 AI Agent 与代码审查，成本降低 77%，服务全球 1/5 网站  
- **Runway 正式成立 Runway Fund 并推出 Builders 计划** [4][5]：双轨并进支持 AI 媒体与世界模拟领域早期公司，覆盖种子轮至 C 轮  
- **Qdrant 推出 Agent Skills 向量检索增强套件** [17]：显著提升智能体系统检索通过率、速度与 Token 效率  
- **Claude Code 源码因 npm map 配置失误泄露** [9]：触发社区对生成式工具供应链安全的深度复盘  
- **Bittensor 完成无许可网络上 72B 参数 LLM 训练** [11]：标志去中心化大模型训练进入工程可行阶段  
- **斯坦福 & MIT 提出 Meta-Harness 智能体外壳自动化框架** [13]：将 NLAHs（自然语言智能体外壳）工程效率大幅提升  
- **智谱 AI 上市后首份财报发布** [21]：全年营收 **7.24 亿元**，领跑国内大模型商业化，MaaS 模式验证高价值闭环
