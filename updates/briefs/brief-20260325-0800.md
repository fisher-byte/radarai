# 3月25日 AI 速报 · 第 144 期

## Summary

OpenAI 正式关停 Sora 独立产品与 API，标志其战略转向核心模型能力聚焦；与此同时，Cursor 发布 Composer 2 技术报告并验证其在 React Native 场景的实用性，而 Perplexity 推出自主智能体 Comet，首次实现端到端浏览器工作流自动化 [14][5][7]。

## Content

## 🔍 核心洞察  
**OpenAI 正式关停 Sora 独立产品与 API**，标志其战略转向核心模型能力聚焦；与此同时，**Cursor 发布 Composer 2 技术报告**并验证其在 React Native 场景的实用性，而 **Perplexity 推出自主智能体 Comet**，首次实现端到端浏览器工作流自动化 [14][5][7]。

## 🚀 重点动态  
- **OpenAI 关停 Sora：战略收缩与产品聚焦** [14]：停止 Sora 独立应用及 API 服务，集中资源强化 GPT 生态与企业级交付。  
- **Cursor 发布 Composer 2 技术报告** [5]：详解训练方法、架构设计与多语言支持能力，获 React Native 开发者实测认可 [22]。  
- **Perplexity 推出 Comet：一款自主互联网计算机** [7]：可执行复杂多步浏览器任务（如比价、表单填写），替代传统外包人力。  
- **Claude Code 引入“自动模式”** [8]：AI 自主决策文件写入与 Bash 命令执行，彻底移除频繁权限确认弹窗。  
- **LangChain 发布 deepagents 0.5.0a1** [2]：支持异步子智能体，使单智能体可并发管理多个后台任务。  
- **LeWorldModel：世界模型的重大突破** [18]：Yann LeCun 团队新架构解决训练不稳定性问题，提升预测一致性与效率。  
- **潜在内省（以及其他开源内省论文）** [9]：研究证实 LLM 具备提示词依赖的**内省能力**，可通过 Logit Lens 检测隐藏概念激活信号。  
- **包管理器需要冷却机制** [12]：Simon Willison 指出依赖“冷却期”是防御供应链攻击的关键工程实践。
