## Content

## 🔍 核心洞察  
**世界模型**智驾首次下放至**8.68 万元级车型**，零跑以极致蒸馏技术打破高阶智驾的算力门槛；**GLM-5.1**编程能力直追**Claude Opus 4.6**，国产大模型在代码推理赛道加速突围 [1][2]；与此同时，**Scion**开源多智能体编排平台、**Accio Work**落地桌面级电商 Agent，标志着 AI Agent 正从概念验证迈向垂直场景深度集成 [18][5]。

## 🚀 重点动态
- **零跑普及世界模型智驾：8.68 万新车实现车位到车位，支持低算力平台** [1]：通过模型蒸馏将具备物理因果理解能力的世界模型部署于入门车型，实现L2+级「车位到车位」智驾。  
- **GLM-5.1 上线，编程表现贴 Opus 4.6 开大，Coding plan 瞬间断货** [2]：智谱发布新版本，强化 reasoning 模式与 IDE/CLI 工具链集成，开发者抢购专业版配额。  
- **华为盘古大模型负责人王云鹤离职，被曝 Agent 创业** [3]：前诺亚方舟实验室主任确认离任，聚焦 AI Agent 方向启动新项目。  
- **Scion 发布：开源多智能体编排平台** [18]：谷歌云团队推出支持容器隔离与 Kubernetes 扩展的轻量级多 Agent 系统框架，已开源并提供探险游戏演示。  
- **Accio Work 桌面 AI Agent 发布** [5]：深度集成 Shopify API，可自动处理订单履约、库存同步与客服话术生成等电商业务闭环。  
- **Stripe 推出 Projects CLI，简化 AI 智能体配置流程** [7]：CLI 工具支持智能体一键注册服务、自动处理 OAuth 流程与密钥轮转，降低工程化门槛。  
- **Roboflow 开源多目标追踪工具** [6]：即插即用型 MOTA/MOTP 优化追踪器，兼容 YOLO/RT-DETR 等任意检测器，提供 CLI 与 Python API。  
- **Runway 推出 Multi-Shot 功能，实现一键生成完整视频** [17]：单次提示词即可输出含分镜、对话、音效与剪辑逻辑的电影化短片，支持景深与光影自动构图。
