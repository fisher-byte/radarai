# 3月31日 AI 速报 · 第 162 期

## Summary

Claude Code 正式集成「电脑操控（Computer Use）」能力，支持 macOS 原生 GUI 交互；Qwen3.5-Omni 全面展示多模态实时能力，覆盖视听编程、语音情感控制与旅行规划等场景；NVIDIA 与 LangChain 宣布深度合作，黄仁勋将亲临 Interrupt 大会共话企业级 AI Agent 战略 [1][4][3]。

## Content

## 🔍 核心洞察  
**Claude Code** 正式集成「电脑操控（Computer Use）」能力，支持 macOS 原生 GUI 交互；**Qwen3.5-Omni** 全面展示多模态实时能力，覆盖视听编程、语音情感控制与旅行规划等场景；**NVIDIA 与 LangChain** 宣布深度合作，黄仁勋将亲临 Interrupt 大会共话企业级 **AI Agent** 战略 [1][4][3]。

## 🚀 重点动态  
- **Claude Code 正式支持电脑操控（Computer Use）** [3]：Anthropic 将该功能集成至 CLI，实现对应用程序与 UI 的直接交互  
- **Claude Code 电脑操控功能登陆 macOS 研究预览版** [1]：用户可通过 `/mcp` 命令启用，支持 SwiftUI、Electron 及无 CLI 的 GUI 工具  
- **Qwen3.5-Omni 演示系列密集发布** [19][20][22][23][24]：涵盖语音情感/语速/音量实时调控、智能打断多轮对话、旅行规划、视听“氛围编程”及带时间戳的视听字幕生成  
- **LangChain 升级 LangSmith Experiments 详情视图** [9]：重构可视化与比较工作流，显著提升 AI Agent 调试效率  
- **Gemini Live 由全新 Gemini 3.1 Flash Live 模型驱动** [10]：谷歌宣布底层模型升级，强化实时响应性能  
- **LlamaIndex 推出 litesearch：本地优先 RAG 检索工具** [11]：基于 LiteParse 与 Bun 构建的高性能 CLI/TUI 文档检索流水线  
- **Genspark CLI 上线 AI 驱动电话拨打功能 “Call for Me”** [12]：支持自动拨号完成预订等真实世界任务  
- **OpenRouter 新增服务层级与 ELO 排名整合** [14][13]：推出 `/fast` 参数加速推理，并在基准页集成 Design Arena ELO 模型评估体系
