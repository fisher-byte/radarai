## 🔍 核心洞察  
AI 创新正加速向**具身智能**与**系统级效率优化**双轨演进：清华团队开源的**Motus**世界模型性能超越硅谷标杆**40%**，小米**MiMo 团队**以**HySparse 架构**实现 KV Cache **减负80%**，而数据格式选择不当竟致 LLM Token 成本激增**740%**——效率瓶颈正从算法层下沉至工程细节。

## 🚀 重点动态
- **清华开源大一统具身世界模型 Motus**：基于 MoT 架构实现“看-想-动”闭环，在 50 项通用任务中性能超 Pi-0.5 **40%**  
- **小米 MiMo 推出 HySparse 混合稀疏注意力架构**：在 80B 模型上实现 KV Cache **减负80%**，显著降低长上下文推理开销  
- **LLM 效率受数据格式严重制约**：XML/Markdown 为最优输入格式，非标准格式可致 Token 成本飙升 **740%**  
- **字节跳动发布 Seedance 2.0 视频生成模型**：支持多模态参考、精准运镜控制与长视频编辑，强化可控生成能力  
- **极简 AI Agent 框架 nanobot 开源**：HKUDS 团队仅用 **4000 行代码**构建异步消息驱动 Agent 系统  
- **Yann LeCun 剖析 AI 创新本质**：强调**提出正确问题**与**长期创造力**远比数学竞赛式解题更关键  
- **MCP 服务器支持 OAuth 注册**：Philipp Schmid 发布技术指南，推动模型上下文协议（MCP）安全集成落地  
- **Codex Windows 版本确认进入发布倒计时**：内部已实现，即将面向开发者正式推出
