## 🔍 核心洞察  
Anthropic 围绕 **Claude Code** 的多重事件持续发酵，既暴露了其在**计费异常**[14]、**源码泄露争议**[17]与**工程文化反思**[4]上的系统性张力，也催生了 **OpenClaude** 等模型无关的开源替代生态 [16]；与此同时，多模态前沿正加速向**统一空间智能**演进，Puffin 与 Falcon Perception 分别以「与相机共思」和**早期融合 Transformer** 架构重构感知范式 [8][0]。

## 🚀 重点动态  
- **PixVerse V6 发布：AI 视频从拼参数转向拼场景与工作流** [3]：爱诗科技新模型显著提升镜头控制、物理真实感与音效合成，并集成团队协作与实时交互功能。  
- **Puffin：统一多模态空间智能大模型——从任意视角理解和生成世界** [8]：ICLR 2026 入选模型，首创将相机几何表征为语言，实现以相机为中心的可控场景理解与生成。  
- **Falcon Perception：0.6B 参数早期融合 Transformer 开放词汇视觉语言模型** [0]：单一主干网络统一视觉与语言，支持开放词汇基础任务。  
- **Claude Code 源码泄露事件技术架构深度解析** [7]：AINews 揭示其复杂记忆系统、子智能体编排机制与细粒度权限框架设计。  
- **Anthropic 官方承认 Claude Code 计费异常：最高多收 20 倍，单次「你好」消耗 13% 额度** [14]：归因于缓存失效 Bug、高峰期配额策略及促销结束。  
- **OpenClaude：模型无关的 Claude Code 开源分支** [16]：支持无缝接入 GPT-4o、DeepSeek、Gemini 等第三方大模型，突破原生闭源限制。  
- **Qdrant：稀疏向量+难负样本挖掘使搜索相关性较 BM25 提升 28%** [10]：第三篇系列技术报告，聚焦向量检索底层优化实践。  
- **WeClaw + MCP 实现语音自动记录待办事项至 Things** [11]：基于 Claude Code 构建端到端自动化工作流，体现 Agent 工具链落地能力。
