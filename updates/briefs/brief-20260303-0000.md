## Content

## 🔍 核心洞察  
**Claude Code** 的 **Computer PTC** 功能正式落地，显著优化智能体执行效率；**Qwen 3.5 小模型系列**（0.8B–9B）实现边缘端高性能突破；**FireRed-OCR** 以2B参数登顶文档解析榜单；**Nano Banana 2** 支持4K超分与多纵横比生成，视觉创作门槛再降低。

## 🚀 重点动态  
- **Anthropic 推出 Claude Code Computer PTC 功能**：通过程序化工具调用在隔离容器中编排执行，降低延迟并抑制上下文膨胀。  
- **阿里巴巴发布 Qwen 3.5 小模型系列**：覆盖 0.8B 至 9B 参数量，在边缘设备保持高推理效率的同时，多项基准媲美超大模型。  
- **小红书开源 FireRed-OCR 模型**：2B 参数，采用三阶段训练+格式约束强化学习，在文档解析任务上超越多数千亿级通用模型。  
- **Google DeepMind 正式推出 Nano Banana 2**：聚焦光效与纹理保真度，支持灵活纵横比及 521px→2K/4K 超分辨率放大，降低专业级视觉创作成本。  
- **MiniMax 发布首份大模型财报**：2025 年收入达 7900 万美元（同比+158.9%），战略转向 AI 平台化，但亏损扩大至 18.7 亿美元。  
- **OpenClaw 连续运行 3 周提炼五大实战经验**：涵盖结构化记忆、成本分层调度、规则工程、主动任务触发与防御性架构设计。  
- **AGENTS.md 文件被证实显著提效**：在编程智能体任务中可缩短运行时间 28%，降低 API 成本 16%，成为轻量级 Agent 工程新范式。  
- **Z.ai 启动 Startup 扶持计划**：为早期初创企业提供免费 API 额度、优先技术支持及新功能早期访问权，加速 AI 应用落地。
