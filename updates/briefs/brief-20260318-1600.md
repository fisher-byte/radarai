# 3月18日 AI 速报 · 第 124 期

## Summary

GPT-5.4 Mini/Nano 与 Claude Cowork Dispatch 的发布，标志着行业正加速向「轻量模型+智能体协同」架构演进；与此同时，Mamba-3、Nemotron 3 Nano 4B 及 FlashAttention-4 等底层技术突破，正系统性提升混合架构效率与边缘部署可行性 [9][10][6][18][13]。

## Content

## 🔍 核心洞察  
**GPT-5.4 Mini/Nano** 与 **Claude Cowork Dispatch** 的发布，标志着行业正加速向「轻量模型+智能体协同」架构演进；与此同时，**Mamba-3**、**Nemotron 3 Nano 4B** 及 **FlashAttention-4** 等底层技术突破，正系统性提升混合架构效率与边缘部署可行性 [9][10][6][18][13]。

## 🚀 重点动态  
- **OpenAI 发布 GPT-5.4 Mini 和 Nano 模型** [9]：定位即时响应代码助手与低成本简单任务，专为多智能体系统中的专用工作者设计  
- **Anthropic 推出 Claude Cowork Dispatch 功能** [10]：支持手机 App 远程调度桌面版 Claude 持续工作，大幅降低非开发者使用门槛  
- **阿里钉钉发布 AI 原生工作平台“悟空（WuKong）”** [7]：底层能力全面 CLI 化，构建强执行力、高安全性的企业级智能体军团  
- **普林斯顿团队发布 FlashAttention-4** [13]：针对英伟达 Blackwell 架构深度优化，将 B200 GPU 算力利用率从 30% 提升至 71%  
- **智谱开源 0.5B 本地 OCR 模型** [15]：免费、轻量、可离线部署，显著降低个人及小公司 AI 应用门槛  
- **LlamaParse 引入视觉锚定机制** [17]：通过边界框引用解决 AI 智能体文档处理中的信任与可审计性难题  
- **Unsloth Studio 上线本地无代码 LLM 微调界面** [20]：显存占用降低 70%，支持消费级硬件高效微调  
- **LibTV 成为首个双入口 AI 视频产品** [3]：同时提供专业 UI 画布与 Agent Skills 接口，兼顾创作者控制力与智能体自动化调用
