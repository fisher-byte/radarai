## Content

## 🔍 核心洞察  
**GPT-5.3-Codex** 正式全面登陆 OpenAI **Responses API** 与 **OpenRouter**，Token 效率提升 **3–4 倍**，并在 Terminal Bench 等多项编程基准中登顶；与此同时，**Anthropic** 推出 **Claude Code 远程控制**功能及全新 **RSP 3.0 负责任扩展政策**，而 **OpenAI** 和 **Perplexity/Comet** 分别以 **WebSocket 加速**和 **gpt-realtime-1.5** 驱动下一代智能体交互体验。

## 🚀 重点动态  
- **GPT-5.3-Codex 全面开放**：OpenAI 通过 Responses API 向所有开发者发布该模型，支持高精度编程与智能体任务编排。  
- **GPT-5.3-Codex 登陆 OpenRouter**：作为先进智能体编程模型，已面向第三方平台开发者开放调用。  
- **Claude Code 推出“远程控制”功能**：支持跨设备（桌面↔移动端）终端会话接力，`claude rc` 命令现已向 **Claude Max 用户**开放研究预览。  
- **OpenAI Responses API 扩展文件支持**：新增原生解析 **docx、pptx、csv、xlsx** 文件能力，强化文档上下文检索精度。  
- **Perplexity & Comet 上线升级版免提语音模式**：基于 OpenAI 全新 **gpt-realtime-1.5** 模型，工具调用稳定性提升 **25%**，支持全浏览器免提控制。  
- **Notion Workers™️ Alpha 发布**：Notion 推出可编程 CPU 基础设施，为平台内定制化 AI 智能体提供底层运行时支持。  
- **OmniDocBench 评测体系被呼吁重构**：Jerry Liu 指出当前文档理解基准已**趋于饱和与过拟合**，亟需转向语义正确性导向的评测标准。  
- **Anthropic 发布 RSP 3.0 及安全资源中心**：更新“负责任扩展政策”，整合全部安全承诺、历史报告与合规实践，强化 AI 安全透明度。
