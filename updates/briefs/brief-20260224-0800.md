## 🔍 核心洞察  
**OpenAI** 全面升级实时能力：发布 **gpt-realtime-1.5** 模型并为 **Responses API** 引入 **WebSockets** 支持，实现最高 **40% 的首字节时间（TTFT）降低**；与此同时，**Anthropic** 提出全新 **“人格选择模型”** 解释 Claude 的类人行为，并指控多家中国实验室实施大规模 **“蒸馏攻击”**。

## 🚀 重点动态  
- **Anthropic 提出“人格选择模型”**：新理论框架用于解释 Claude 等 AI 助手为何呈现稳定、类人语言风格与价值观对齐行为。  
- **OpenAI 发布 gpt-realtime-1.5**：显著提升指令遵循、多语言处理（含复杂口音）及语音工作流质量，已达业界领先（SOTA）水平。  
- **OpenAI Responses API 正式支持 WebSockets**：通过持久连接与增量输入，使 AI 智能体 TTFT 最高降低 **40%**，延迟整体下降 **20–40%**。  
- **Cursor 与 Vercel AI SDK 已集成 WebSockets**：分别实现 **30%** 与 **40%** 的响应速度提升，加速开发者工具链与智能体工作流。  
- **Glean 助手集成 gpt-realtime-1.5**：打造亚秒级延迟、支持企业数据实时接入的语音助手，强化办公场景落地能力。  
- **Anthropic 指控 DeepSeek 等三家公司实施“蒸馏攻击”**：称其通过数千万次对话非法提取 Claude 能力训练竞品模型。  
- **OpenAI 弃用 SWE-bench Verified，转向 SWE-bench Pro**：因严重数据污染与基准饱和，启用更鲁棒的新一代编程评估标准。  
- **Google Gemini Interactions API 推出原生多模态函数调用**：支持智能体直接处理图像输入并生成真实图像输出，拓展多模态 Agent 边界。
