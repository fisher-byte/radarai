## 🔍 核心洞察  
**Codex 应用首周下载破 100 万**，**OpenAI GPT-5.3-Codex** 已在 Cursor、GitHub 与 VS Code 全面上线；与此同时，**Claude Opus 4.6** 登顶 LMArena 代码与文本双榜，而 **Perplexity Deep Research** 与 **Cursor Composer 1.5** 亦同步升级至同代引擎——AI 编程基础设施正加速进入「多模型协同、全工具嵌入」新阶段。

## 🚀 重点动态
- **OpenAI Codex 应用首周下载量突破 100 万**：用户增长率达 **60%**，官方确认长期保留免费层级。  
- **GPT-5.3-Codex 正式登陆主流开发工具**：即日起在 **Cursor、GitHub 和 VS Code** 全面可用，API 发布因网络安全标准要求暂缓。  
- **Claude Opus 4.6 荣登 LMArena 双榜首**：在 **Code Arena** 与 **Text Arena** 基准测试中均位列第一。  
- **Perplexity Deep Research 升级至 Opus 4.6**：显著提升 Google DSQA 等专业基准表现，面向 Max/Pro 用户开放。  
- **Cursor 发布 Composer 1.5**：集成全新 **1T 参数以下全球最强代码模型**，聚焦智能水平与响应速度的平衡。  
- **Google 推出 Embedding Gemma 微调实验室**：在 Hugging Face 上线交互式 Space，支持开发者定制嵌入模型与个性化排序。  
- **DeepMind 发布 Perch 2.0 水下生物声学模型**：实现从鸟类鸣叫到水下声学的跨域泛化，已开放端到端科研 Demo。  
- **SE-Bench 新基准发布**：首个专注评估模型「知识内化」与**自我进化能力**的开源评测框架。
