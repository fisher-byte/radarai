# 3月25日 AI 速报 · 第 143 期

## Summary

MCP 协议、GUI-Agent 架构与离线评估框架正成为 AI 智能体工程化落地的关键技术支点；Figma × Claude Code深度集成、Replit Agent 4 Buildathon超 3000 人参与，标志智能体开发生态加速成熟 [5][2][10]。

## Content

## 🔍 核心洞察  
**MCP 协议**、**GUI-Agent 架构**与**离线评估框架**正成为 AI 智能体工程化落地的关键技术支点；**Figma × Claude Code**深度集成、**Replit Agent 4 Buildathon**超 3000 人参与，标志智能体开发生态加速成熟 [5][2][10]。

## 🚀 重点动态
- **Figma 通过 MCP 集成 Claude Code** [5]：发布新 MCP 工具，使 AI 智能体可基于设计系统上下文直接在画布执行设计操作  
- **Replit 正式启动 Agent 4 Buildathon** [2]：吸引超 3,000 名开发者参赛，总奖金池达 5.7 万美元  
- **生产就绪的 LLM 智能体：离线评估的综合框架** [10]：提出路由、LLM-as-Judge、RAG 评估三支柱框架，支持自动化验证智能体稳定性  
- **【GUI-Agent】阶跃星辰 GUI-MCP 解读——整体架构** [11]：详解 GUI-MCP 标准化接口、分层架构与模型分发策略，推动 GUI 自动化任务标准化  
- **英伟达发布 Kimodo 模型：通过提示词实现 3D 动作生成** [16]：基于 700 小时动捕数据训练，支持人形与机器人骨骼动作控制  
- **Google DeepMind 与 Agile Robots 达成合作** [20]：将 Gemini 基础模型集成至实体机器人硬件，加速具身智能落地  
- **见证物理世界的觉醒：EAI-100 具身智能年度榜单发布** [18]：魔搭联合发布 2025 年中国具身智能百项代表性成果与人物白皮书  
- **LangChain 将在纽约举办 AI 智能体工作坊** [0]：4 月 16 日线下聚焦生产级智能体工作流与工具调用实战
