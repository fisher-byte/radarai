## 🔍 核心洞察  
**SWE-1.6** 成为本周最强技术信号：**Cognition Labs** 与 **Windsurf** 先后发布该模型早期预览版，其在 **SWE-Bench Pro** 基准上全面超越 SWE-1.5 及当前顶尖开源模型；与此同时，**Clay** 实现对每月 **3 亿次智能体运行** 的全链路可观测性，标志着 AI Agent 工程化进入规模化运维新阶段。

## 🚀 重点动态  
- **Cognition Labs 发布 SWE-1.6 预览版**：在 SWE-Bench Pro 上性能跃升，推理速度与准确率同步突破  
- **Windsurf 推出 SWE-1.6 早期预览版**：强调极高的处理吞吐与低延迟响应能力  
- **Clay 基于 LangSmith 管理 3 亿次/月智能体运行**：实现调试、成本对账与自动化模型评估三位一体可观测性  
- **“技能注入”被确认为新型 AI Agent 安全威胁**：恶意第三方 Skill 文件可绕过指令防护实施攻击  
- **Claude 上线 ChatGPT 数据导入功能**：支持历史对话与上下文无缝迁移，强化跨平台用户粘性  
- **Cognition 训练架构优化提速 6 倍**：通过容忍高陈旧度算法与异步 RL 显著提升 GPU 利用率  
- **Perplexity Computer 自主构建 5000 行宝可梦金融应用**：端到端完成调研、编码、调试与部署闭环  
- **Qdrant 发布轻量级相关性反馈搜索教程**：无需重训模型，即刻优化 RAG 与语义搜索质量
