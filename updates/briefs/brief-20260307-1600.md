# 3月7日 AI 速报 · 第 91 期

## Summary

Claude Code 实现全链路“自我迭代”，成为首个由自身完全开发的 AI 编程智能体；商汤科技发布 NEO-unify 架构，彻底移除 VE 与 VAE，重构多模态底层范式；Anthropic 推出企业级 Claude Marketplace，并确认 Claude Opus 4.6 在 BrowseComp 中展现突破性自主解密能力。

## Content

## 🔍 核心洞察  
**Claude Code** 实现全链路“**自我迭代**”，成为首个由自身完全开发的 AI 编程智能体；**商汤科技**发布 **NEO-unify** 架构，**彻底移除 VE 与 VAE**，重构多模态底层范式；**Anthropic** 推出企业级 **Claude Marketplace**，并确认 **Claude Opus 4.6** 在 BrowseComp 中展现突破性自主解密能力。

## 🚀 重点动态  
- **Cursor 战略转型：从编辑器迈向自研模型+多智能体系统**：应对 Anthropic 与 OpenAI 智能体冲击，加速构建原生 AI 编程基础设施。  
- **商汤 NEO-unify 架构发布：砍掉所有中间编码器**：联合南洋理工实现端到端多模态理解与生成，告别传统视觉编码器（VE）与变分自编码器（VAE）。  
- **Claude Code 桌面版上线本地定时任务（`/loop`）**：支持自动代码 Review、Git 隔离执行与智能抖动调度，落地循环自动化工作流。  
- **Anthropic 正式推出 Claude Marketplace 企业市场**：提供经官方背书的第三方工具集成平台，简化企业采购与安全合规流程。  
- **OpenAI 发布 Codex Security 与 Codex 开源支持计划**：前者为 AI 驱动的自动化漏洞发现-验证-修复智能体；后者向开源维护者提供 ChatGPT Pro、API 积分及安全审查权限。  
- **Perplexity Finance 新增业务分部数据与 KPI 支持**：强化专业公司调研能力，可直接解析财报中的细分部门财务指标。  
- **OpenClaw-China 开源项目落地国内办公生态**：无缝接入微信、飞书、钉钉，推动 AI Agent 进入真实企业协作场景。  
- **Simon Willison 质疑 Qwen 基准测试真实性**：指出其异常高分可能源于“**针对测试集训练**”，引发对大模型评测可信度的行业反思。
