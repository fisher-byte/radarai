## Content

## 🔍 核心洞察  
**Anthropic** 完成 **300 亿美元融资**、估值飙升至 **3800 亿美元**，其 **Claude Code** 周活用户翻倍成关键驱动力；与此同时，**Gemini 3 Deep Think**、**GPT-5.3-Codex-Spark** 与 **DM0（2.4B具身原生模型）** 等新一代**轻量化+强推理+高闭环**模型密集落地，标志着AI正从“大而全”加速转向**可部署、可审计、可闭环**的工程化新阶段。

## 🚀 重点动态  
- **Letta 发布 Context Repositories**：首创基于 **Git 的 Agent 记忆管理范式**，实现记忆的可追溯、可并发与人类可审计。  
- **Anthropic 获 300 亿美元融资**：估值达 **3800 亿美元**，**Claude Code** 用户激增与网页版多仓库/Git 可视化升级成核心支撑。  
- **原力灵机发布 DM0 模型**：仅 **2.4B 参数**，通过**空间推理思维链**与**全身全时全域数据闭环**，在轻量化硬件上实现物流场景高频实时控制。  
- **MiniMax 推出 M2.5 旗舰模型**：10B 小参数实现**全栈编程+智能体能力**，吞吐达 **100TPS**，宣称“1 美金时薪雇全栈替身”。  
- **Google 上线 Gemini 3 Deep Think 推理模式**：支持**草图→3D 打印模型**等复杂生成任务，并同步发布官方 **Gemini API 技能仓库**（GitHub）。  
- **Cognition 开放 GPT-5.3-Codex-Spark 预览版**：登陆 Windsurf 竞技场，Warp 终端已集成，OpenAI 强调其**毫秒级响应与迭代效率跃升**。  
- **Replit Agent 新增用户直连反馈闭环**：终端用户可一键提交 Bug 与需求，驱动 Agent 自动化调试与版本迭代。  
- **Tambo 开源框架发布**：推动 **Generative UI（生成式 UI）** 落地，让 React 应用摆脱聊天框限制，实现 AI 动态调用组件的**真正 AI 原生交互**。
